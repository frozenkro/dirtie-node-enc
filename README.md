# Dirtie Node Enclosure

Enclosure design for (dirtie-node)[https://github.com/frozenkro/dirtie-node]

Export this to .stl using openscad.

## Components

- Raspberry Pi Pico W (or Pico 2 W)
- Adafruit STEMMA Capacitive Soil Sensor
- PKCELL LP503562 3.7V 1200mAh LiIon Battery
- Lipo Amigo LiIon Charging Module
